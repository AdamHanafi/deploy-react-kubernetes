# React-redux-api-example

This project calls the OMDb API to get information about a movie the user entered using React and Redux. 

Tools used: 
- React
- Redux
- OMDB API 
- Webpack
- Babel
- Sass 

## How to run the project 

In the project directory run: 

`npm install`

`npm run build-css` 

`npm run start:dev`
